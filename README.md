Aplikasi web untuk membantu pengelolaan warung, mulai dari manajemen produk, transaksi, hingga laporan penjualan.

## ✨ Fitur

* Login & Logout
* Dashboard
* Manajemen Produk
* Manajemen Kategori
* Transaksi Penjualan
* Laporan Penjualan
